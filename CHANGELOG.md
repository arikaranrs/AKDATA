# Changelog

All notable changes to the AKDATA library will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-27

### Added
- Initial stable release of the AKDATA preprocessing framework.
- Core pipeline engine supporting custom fit/transform steps.
- Safe IO loaders supporting CSV, Excel, and JSON files.
- EDA, profiling, skewness/kurtosis, and Dataset Health Score metrics.
- Preprocessing steps: cleaning, missing values, duplicates, outliers, datatype casting, encoding, scaling, and math transformations.
- Target leakage detection and Schema validation checking.
- Summary reporting, HTML interactive dashboards, and PDF exporters.
- End-to-end Orchestrator for simple automated workflow preparation.
