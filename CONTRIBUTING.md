# Contributing to AKDATA

Thank you for your interest in contributing to AKDATA! We welcome contributions from data scientists, machine learning engineers, students, and open-source enthusiast developers.

## How to Contribute

1. **Fork the Repository**: Create a personal fork on GitHub.
2. **Clone the Fork**: Clone your fork locally:
   ```bash
   git clone https://github.com/your-username/akdata.git
   cd akdata
   ```
3. **Set Up Development Environment**:
   ```bash
   pip install -r requirements.txt
   pip install -e .
   ```
4. **Create a Branch**: Create a feature branch:
   ```bash
   git checkout -b feature/my-new-feature
   ```
5. **Implement Changes & Tests**: Ensure all existing tests pass and write new tests covering your additions.
6. **Run Tests**:
   ```bash
   pytest
   ```
7. **Commit & Push**: Commit your changes following PEP8 rules, push to your fork, and submit a Pull Request.
